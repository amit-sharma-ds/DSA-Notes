Maximum sub array sum (using BF approach with a optimization)
```
#include <iostream>
#include <climits>
using namespace std;

int main() {
    int n = 5;
    int arr[5] = {1, 2, 3, 4, 5};
    
    int maxSum = INT_MIN;

    for(int st = 0; st < n; st++) {
        int currSum = 0;
        for(int end = st; end < n; end++) {
            currSum += arr[end];
            maxSum = max(currSum, maxSum);
        }
    }

    cout << "max subarray sum = " << maxSum << endl;

    return 0;
}
```

ii. Logic/ Intuition
- To kadane's algorithm kis logic ke upr, kis intution ke upr work krti hai, pehle uski bat krte hai.
- To kadane's algorithm jis intuition ke upr work krti hai, wo basically 2 no.s ka sum  calculate krne ka intuition hai.

- +ve + +ve --> + : agr hmare pass already koi +ve number hai, usme hm +ve ko add krenge. to hme pta hai, final answer +ve ayega.
- -ve + large +ve --> + : agr hmare pass koi chota -ve number hai, usme hm koi bda -ve number add krenge. to bhi answer +ve ayega.
- `chota +ve + bda -ve --> -ve: lekin hmare pass koi chota +ve number hai, jisme ander hm koi bda -ve no. add krenge. To hmara number -ve aane wala hai. To kadane's Algorithm isi logic ke upr work krti hai.`

- Basically, agr hmare pass already kuch sub arrays ka sum +ve aa rha hai, ya let's suppose chota -ve bhi aa rha hai. Us sum me hm ek bda -ve add nhi krna chahenge.
- Kuki jb bhi koi +ve value hai, uske andr -ve add hoga, to usse hmesha value decrease hogi, usse kbhi bhi maximum sub array sum nhi aa skta.
- reset to 0: To kbhi bhi is your sub array sum becomes -ve. To kadane's algorithm kehta hai, use reset kr do to 0.
- Kuki sub array sum ke andr ek -ve value se acha hai, ki hm 0 value ko add kr de.

```
class Solution {
    public:
        int maxSubArray(vector<int>& nums) {
            int currSum = 0, maxSum = INT_MIN;

            for(int val : nums) {
                currSum += val;
                maxSum = max(currSum, maxSum);

                if(currSum < 0) {
                    currSum = 0;
                }
            }

            return maxSum;
        }
};
```

- Yha pr instead of array ek vector given hai.
- sbse pehle current sum and max sum likha.
- for each loop on vector
- currsum += val: means current sum equal to this value
- maxSum: maxSum calculate krenge. max of current-sum and maxSum
- if currSum < 0, currSum = 0: reset wala kam krenge, agr hmare currentSum ki value less than 0 hai, to us case me currSum ko reinitialize kr denge 0.
- return maxSum: and finally last me hm return kr denge apne maxSum ko as our final answer.
